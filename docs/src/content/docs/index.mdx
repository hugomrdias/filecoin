## Features

<CardGrid stagger>
	<Card title="Core Primitives" icon="seti:javascript">
		It provides core utilities, abstractions and types for primitives such as: RPC, Signature, Address, Token, Chain, Wallet and more. 
	</Card>
	<Card title="Ledger Support" icon="seti:lock">
		Provides an API for the Filecoin Ledger embedded app. 
	</Card>
	<Card title="React Hooks" icon="seti:react">
		React hooks for wallet, account, balance, transations, signing and more.
	</Card>
	<Card title="Wallet Adapters" icon="seti:default">
		Connect to all the Filecoin Wallets with one single interface, support for Filsnap, Ledger and more.
	</Card>
	<Card title="AI support" icon="magnifier">
		Index this website with your editor and add to your LLM context easily with `llms.txt` support.
	</Card>
</CardGrid>
